Customer Retention & Churn Analysis

Project Overview
This project analyzes customer churn behavior for a telecom subscription-based business using Power BI. The objective is to identify churn patterns, retention drivers, customer lifetime trends, and provide actionable business recommendations to reduce customer loss.


Internship Task
Task 2 – Customer Retention & Churn Analysis


Tools & Technologies Used
- Power BI
- Power Query
- DAX
- Microsoft Excel / CSV Dataset


Dataset Information
The dataset contains telecom customer information including:
- Customer demographics
- Subscription details
- Internet services
- Payment methods
- Monthly charges
- Churn status


Key KPIs
- Total Customers
- Churn Customers
- Churn Rate %
- Retained Customers
- Average Monthly Charges
- Customer Lifetime Value (CLV)


Dashboard Features
- Customer churn analysis
- Contract-wise churn trends
- Tenure-based retention analysis
- Payment method analysis
- Internet service churn comparison
- Customer segmentation insights
- Interactive slicers and filters


Key Insights
- Month-to-month customers show the highest churn rate.
- Customers with higher monthly charges are more likely to churn.
- Long-tenure customers demonstrate stronger loyalty.
- Fiber optic users contribute to elevated churn levels.
- Electronic check users show higher churn behavior.


Business Recommendations
- Promote long-term contracts through discounts and loyalty benefits.
- Improve onboarding experience for new customers.
- Optimize pricing strategies for high-risk customers.
- Encourage automatic payment methods.
- Improve customer support for fiber optic users.


Project Outcome
The dashboard helps identify high-risk customer segments and provides data-driven recommendations to improve customer retention and reduce churn.
